Macro Definition Documentation

◆ CY_QI_CEP_TIMEOUT_NOISE_RETRY_CNT

#define CY_QI_CEP_TIMEOUT_NOISE_RETRY_CNT   (2u)

Number of times to retry CEP timeout in event of noise.

This is done to avoid CEP timeout due to noisy phones.

◆ CY_QI_BMC_RX_FREQ

#define CY_QI_BMC_RX_FREQ   (2000u)

BMC RX baud rate.

This is 2KHz as defined by Qi specification.

◆ CY_QI_BMC_RX_BIT_SPI_OVER_SAMPLE

#define CY_QI_BMC_RX_BIT_SPI_OVER_SAMPLE   (8u)

This field indicates the over sample rate to be used.

It can only take 8 or 16.

◆ CY_QI_BMC_RX_SPI_RAW_DATA_SIZE

#define CY_QI_BMC_RX_SPI_RAW_DATA_SIZE   (64u * (CY_QI_BMC_RX_BIT_SPI_OVER_SAMPLE))

Assuming a preamble of 25 bits, (1 + 27 + 1) bytes at 11 bits per byte, there will be about 44 bytes.

Since we do not get to stop at the exact boundary, assuming 64 byte buffer is sufficient to hold all possible raw input. Each byte is stored in 8X scale or 16X scale.

◆ CY_QI_COIL_FB_HB_SWITCH_FREQ

#define CY_QI_COIL_FB_HB_SWITCH_FREQ   (2750u)

145 Khz is default; x2 because of the divider circuit .

Remove 2% from Clock accuracy on worst corners so 142.5Khz
