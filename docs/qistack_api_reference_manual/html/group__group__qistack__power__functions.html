cy_en_qi_status_t Cy_QiStack_PID_Trigger ( cy_stc_qi_context_t *  qiCtx,
int8_t  controlError 
)

This function trigger PID algorithm for control loop correction.

Parameters
qiCtxQiStack Library Context pointer.
cepControl Error value for PID algorithm.
Returns
CY_QISTACK_STAT_SUCCESS if operation is successful CY_QISTACK_STAT_BAD_PARAM if the input parameters are not valid

1mS time elapse comes from timer start. Hence, consider 1mS as 0mS.

Stop the timer to make sure dangling timer is not running.

Run PID correction only if the coil voltage is stable. If previous correction has still not completed, likely case of load dump, then we should skip ID correction.

Also Run PID loop only for non-zero control error
